# Simple Notepad

A lightweight, browser-based plain text editor built with vanilla HTML, CSS, and JavaScript.

The application focuses on simplicity and minimalism, providing a distraction-free environment for quick notes and plain text editing.

## Features
- Plain text editing in the browser
- Supports the modern File System Access API where available
- Common keyboard shortcuts (Ctrl+S, Ctrl+F, etc.)
- No external libraries or frameworks

## Demo
Live demo via GitHub Pages:
https://johnfullerroot.github.io/simple-notepad/

## Usage
- Open the app in a modern browser
- Start typing to create or edit plain text
- Use **Ctrl+S** to save the file (where supported)
- Use standard browser find (**Ctrl+F**) to search text

For best results, use a browser that supports the File System Access API.
